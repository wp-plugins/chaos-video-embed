== Description ==

This plugin makes it easy for you to embed media from ChAOS in your wordpress site. 

== Installation ==


1. Upload folder `mcm-plugin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add video to post via "Chaos Video embed" panel in Post
